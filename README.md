# What is this
My emacs [doom](https://github.com/doomemacs/doomemacs) setting

## How do I use it
Note:
+ install emacs doom
+ from the `:confi` settings in [init.el](./init.el) activate the `literate` mode
+ modify the [package.el](./package.el)
+ copy the [config.org](config.org)
+ execute 

```
~/.config/emacs/bin/doom sync
~/.config/emacs/bin/doom doctor
```

Note:
if `doom` is not invoked, remove `~/.emacs` file/directory and try again.
